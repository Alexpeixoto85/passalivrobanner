Configure o Firebase

Cole o firebaseConfig no topo deste arquivo, igual fez no portal.

Este painel precisa, além do que o portal já usa:

  1. Ativar o provedor E-mail/senha em Authentication → Sign-in method.
  2. Criar o seu usuário administrador (na primeira vez, use o botão Criar conta de administrador nesta tela).
  3. Publicar as regras do Firestore para a coleção banners (veja abaixo), trocando pelo seu e-mail.
